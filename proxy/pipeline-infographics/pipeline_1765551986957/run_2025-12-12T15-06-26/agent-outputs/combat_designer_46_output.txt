# Combat System Design Document
## Warhammer 40,000: Imperium Divided - Damocles Gulf Sector

---

# I. COMBAT PHILOSOPHY

Drawing from Guild Wars 1's combat design, this system emphasizes **skill-based tactical combat** over stat-checking and gear grinding. Combat should feel weighty, meaningful, and distinctly Warhammer 40k.

**Core Principles:**
- Build diversity > single meta builds
- Positioning and timing > button mashing
- Team composition matters (8-player parties)
- Every weapon type has a role
- Death is punishing but not devastating
- Combat tells stories of the grimdark future

---

# II. COMBAT MECHANICS OVERVIEW

## A. Combat System Type: Real-Time with Tactical Pause Elements

```
COMBAT FLOW

┌─────────────────────────────────────────────────────────────┐
│                    REAL-TIME COMBAT                         │
│                                                             │
│  Movement ←→ Ability Use ←→ Auto-Attack ←→ Positioning     │
│       ↓           ↓             ↓              ↓            │
│   Dodge/Cover   Cooldowns   Energy Regen   Line of Sight   │
│       ↓           ↓             ↓              ↓            │
│   Damage Taken  Effects     DPS Output    Target Priority  │
└─────────────────────────────────────────────────────────────┘

TACTICAL ELEMENTS (Guild Wars 1 Style):
- 8 skill slots (limited from larger pool)
- Skills must be equipped before mission
- No skill swapping mid-combat
- Party composition critical
- Hero/Henchman AI companions available
```

```
------------|----------------------------|-------------------|---------------
```

## B. Damage Calculation Formula

```
DAMAGE FORMULA (Guild Wars 1 Inspired)

Base Damage = Weapon Damage × (1 + Attribute Bonus)

Modified Damage = Base Damage 
                  × Skill Multiplier 
                  × Critical Multiplier (if crit)
                  × Type Effectiveness
                  × (1 - Damage Reduction)

DAMAGE REDUCTION FORMULA

Armor DR = Armor / (Armor + 60)
├── 60 Armor = 50% reduction
├── 120 Armor = 67% reduction  
├── 180 Armor = 75% reduction
└── Cap: 85% reduction (340 Armor)

CRITICAL HIT CALCULATION

If Random(0,100) < (Base Crit% + Modifiers):
    Apply Critical Damage Multiplier
    
Critical Damage = Base × (1.5 + Crit Damage Bonuses)

TYPE EFFECTIVENESS

Strong Against: ×1.25 damage
Neutral: ×1.00 damage
Weak Against: ×0.75 damage
```

### Example Damage Calculation

```
EXAMPLE: Fire Warrior Pulse Rifle Shot

Weapon Base Damage: 28
Marksmanship Attribute: 10 (+30% ranged damage)
Skill Used: Aimed Shot (×1.4 multiplier)
Target: Ork Boy (Light Armor, weak to Plasma)

Calculation:
Base = 28 × 1.30 = 36.4
Skill Modified = 36.4 × 1.4 = 50.96
Type Bonus = 50.96 × 1.25 = 63.7
Ork Armor (45) DR = 45/(45+60) = 0.43
Final Damage = 63.7 × (1 - 0.43) = 36.3

Result: 36 damage dealt
```

## B. Skill System (Guild Wars 1 Style)

```
SKILL SLOT STRUCTURE

[1][2][3][4][5][6][7][8] + [Elite Slot]

Slot 1-2: Primary Profession Skills
Slot 3-4: Primary OR Secondary Profession Skills
Slot 5-6: Secondary Profession Skills
Slot 7-8: Universal/Faction Skills
Elite: Powerful signature ability (one per bar)

SKILL TYPES

Attack Skills: Direct damage abilities
├── Cost: 5-15 Energy
├── Cooldown: 3-15 seconds
├── Examples: Bolter Burst, Pulse Volley, Choppa Swing

Support Skills: Buffs, heals, utility
├── Cost: 5-25 Energy
├── Cooldown: 10-45 seconds
├── Examples: Litany of Hate, Bonding Knife, WAAAGH! Shout

Control Skills: CC, debuffs, positioning
├── Cost: 10-20 Energy
├── Cooldown: 15-30 seconds
├── Examples: Suppressing Fire, Markerlight, Intimidate

Elite Skills: Powerful signature abilities
├── Cost: 15-25 Energy
├── Cooldown: 45-120 seconds
├── Acquisition: Quest rewards, boss drops, faction rank
├── Examples: Orbital Strike, Mont'ka Protocol, WAAAGH! Storm
```

```
   ├── Current: Melee deals ~20% more DPS, takes ~40% more damage

   └── Status: Needs tuning pass

   ├── Current: 40 base energy, 3/sec regen
   ├── Risk: Builds that generate infinite energy
   ├── Mitigation: Energy generation abilities capped at +50% regen
   └── Status: Monitor for exploits

   ├── Current: 1-4 second hard CC
   ├── Risk: Perma-CC chains
   ├── Mitigation: CC applies diminishing returns (50% reduction on reapply within 10s)
   └── Status: Implemented

4. HEALING THROUGHPUT
   ├── Current: ~5% HP/second sustainable healing
   ├── Risk: Immortal tank builds
   ├── Mitigation: Healing reduction debuff exists, anti-heal bosses
   └── Status: Acceptable
```

This combat system delivers Guild Wars 1-style tactical depth within the Warhammer 40,000 universe:

**Core Strengths:**
- Build diversity through profession combinations

- Faction-appropriate combat feel (Imperial faith, Tau coordination, Ork brutality)

- Tunable formulas for ongoing balance

**Key Numbers:**
- 8 skill slots per loadout
- 40 base energy, 3/sec regeneration
- 480 base health
- 85% armor damage reduction cap
- 1-4 second CC durations with diminishing returns
- 5-15 minute boss encounters for 8-player parties

**Balance Philosophy:**
- No single dominant strategy
- All playstyles viable
- Consumables enhance but don't trivialize
- Gear matters but skill matters more
