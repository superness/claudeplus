The tests were written expecting a 2-player mode where the turn would stay on "Black's Turn" waiting for user input. **This is not a bug** - the AI opponent feature is working as designed. The tests need to be updated to account for AI behavior, or the game needs a toggle to disable AI for testing purposes.
