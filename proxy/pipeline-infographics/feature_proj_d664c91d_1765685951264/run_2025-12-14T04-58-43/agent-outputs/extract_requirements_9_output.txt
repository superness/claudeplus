```
CHARACTER CREATION FLOW:

STEP 1: FACTION SELECTION (30-60 seconds)
├── Goal: Choose Imperial, Tau, or Ork
├── UI: Three large faction banners with lore
├── Preview: Rotating faction representative models
└── Output: faction_code selected

STEP 2: CLASS SELECTION (45-90 seconds)
├── Goal: Choose 1 of 4 classes for selected faction
├── UI: Four class cards with roles and preview videos
├── Preview: Selected class idle animation + weapon
└── Output: class_code selected

STEP 3: APPEARANCE CUSTOMIZATION (60-120 seconds)
├── Goal: Customize face, hair, skin, scars, augments
├── UI: Dropdown/slider controls + live preview
├── Preview: Close-up character face + full body toggle
└── Output: appearance object populated

STEP 4: CHARACTER NAMING (30-45 seconds)
├── Goal: Enter unique character name
├── UI: Text input with real-time validation
├── Preview: Name displayed above character
└── Output: character_name validated

STEP 5: CONFIRMATION (15-30 seconds)
├── Goal: Review all choices, confirm creation
├── UI: Summary panel with all selections
├── Preview: Final character rotating in victory pose
└── Output: POST /api/v1/characters → game world

TOTAL TIME: 3-5 minutes (as designed in player-experience.md)
```

#### V. Character Naming Validation

**From character-creation-ui-design.md (lines 940-980):**

```
CHARACTER NAME VALIDATION:
├── Regex: ^[A-Z][a-zA-Z\s'-]{2,29}$
├── Min Length: 3 characters
├── Max Length: 30 characters
├── First Character: Must be capital letter (A-Z)
├── Allowed Characters: Letters (A-Z, a-z), space, hyphen, apostrophe
├── Spaces: Allowed, but not leading/trailing
├── Case: Mixed case allowed
└── Unicode: English characters only (for launch)

FORBIDDEN PATTERNS:
├── Reserved Words: Admin, Moderator, GM, Dev, Support, System, Server, Official (case-insensitive)
├── Profanity Filter: Applied (comprehensive list)
├── No repeated special chars: "O''Rourke" ✗, "O'Rourke" ✓
├── No leading/trailing spaces: " Name " ✗, "Name" ✓
└── No Leet speak: "xXN4m3Xx" ✗

UNIQUENESS CHECK:
├── Endpoint: Real-time check via API
├── Scope: Unique within account (not globally unique)
├── Debounce: 500ms after last keystroke
├── Feedback: Immediate visual indicator
└── Error: "Name already exists on your account"

EXAMPLES (Valid):
├── Imperial: "Brother Titus", "Sister Mirael", "Sergeant Harker"
├── Tau: "Shas'ui Vior'la", "Por'el Ka'mais", "Farsight"
├── Ork: "Warboss Grimskull", "Grotfang", "Snikrot"
```
