### How to Use

```bash
cd client

# Run all tests (headless)
npm test

# Run with browser UI for debugging
npm run test:ui

# Run with visible browser
npm run test:headed

# Debug specific test
npm run test:debug
```

In tests, access hooks via:
```typescript
await page.evaluate(async () => {
  const hooks = (window as any).__TEST_HOOKS__;
  const enemyId = await hooks.spawnTestEnemy('CHAOS_CULTIST', 5, 5);
  hooks.selectEnemy(enemyId);
  const result = hooks.useSkill(0);
  return hooks.getEnemyHealth(enemyId);
});
```

### Integration Notes

- The `CombatManager` events are captured automatically via the `onCombatEventCallback`
- Tests automatically start the Vite dev server if not running
