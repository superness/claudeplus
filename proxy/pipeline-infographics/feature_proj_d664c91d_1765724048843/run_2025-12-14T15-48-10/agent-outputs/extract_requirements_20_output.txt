├── Success: Resume loading
├── Failure: "Failed to enter zone" + Return to Character Select
└── Character State: No changes (not yet in zone)

├── Detection: API returns 401
├── Action: Silent refresh using refresh_token
├── Success: New access_token, retry failed request
├── Failure: Prompt for re-login
└── User Impact: Seamless if refresh succeeds

REFRESH TOKEN EXPIRED:
├── Detection: Refresh attempt returns 401
├── Display: Modal "Session expired. Please log in again."
├── Action: Clear stored tokens
├── Navigation: Return to Login Screen
└── User Impact: Must re-enter credentials
